Sleep Health & Lifestyle Analysis Dashboard
Project Overview

This project analyzes the relationship between sleep patterns and lifestyle factors using Power BI. The goal is to identify how variables such as stress levels, physical activity, BMI, and occupation influence sleep quality and sleep duration.

The dashboard provides clear insights through interactive visualizations that help understand patterns in sleep health and lifestyle behaviors.

Dataset

Dataset Name: Sleep_health_and_lifestyle_dataset

The dataset contains information about individuals' sleep patterns, health indicators, and lifestyle habits.

Tools & Technologies

Power BI – Dashboard development and data visualization

Excel / CSV – Data source

Key Performance Indicators (KPIs)

The dashboard highlights the following KPIs:

Average Sleep Duration

Average Sleep Quality Score

Most Common Sleep Disorder

Average Stress Level

Average Daily Steps

These KPIs help summarize overall sleep health and lifestyle behavior.

Dashboard Features

The Power BI dashboard includes several visualizations to analyze relationships in the data:

Age Distribution by BMI Category

Stress Level vs Heart Rate

Daily Steps by Gender

Sleep Quality vs Age

Heart Rate Comparison by Gender

Occupation Distribution

The dashboard is interactive and allows users to explore insights across different groups.

Key Insights

Higher stress levels may negatively impact sleep quality.

Physical activity (daily steps) can influence sleep health.

Certain occupations show higher stress levels and sleep issues.

BMI categories may affect overall sleep patterns.
